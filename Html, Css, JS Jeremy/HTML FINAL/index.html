What is a Rubik's Cube?

The Rubik's Cube is a 3D combination puzzle invented in 1974 by Hungarian sculptor and professor of architecture Ern≈ë Rubik. It consists of a cube made up of smaller cubes, each of which can be rotated along multiple axes. The goal of the puzzle is to restore the cube to its original configuration, with each side displaying a single color.
